Hey! I'm Max, welcome to my website. I am a third-year Computer Science student at Cal Poly, San Luis Obispo, with a passion for programming and design. I created this website to showcase some of the projects I've worked on in these areas.

ASCII Video Generator

I initially wrote a Python program that created a video-like effect by rapidly printing "frames" as blocks of text in the terminal, sourcing data from a pre-recorded video. Later, I developed a JavaScript version that writes characters on a video canvas on an HTML webpage based on live webcam footage

TerminalStats

This C program elevates the macOS terminal experience with ASCII art and real-time system stats using various system information retrieval methods. Get it here!

Type & Color

I created this page as a simple way to play around with typefaces and foreground/background color combos, inspired by a print publishing class I took and Sanzo Wada's A Dictionary of Color Combinations.
